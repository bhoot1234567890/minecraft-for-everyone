build Troubleshooting

"Failed to connect to server" expand_more
  • • Double-check the server address
  • • Server might be offline - contact admin
  • • Try disabling your VPN
  • • Check firewall settings
"Disconnected" immediately after joining expand_more

This usually means you're not whitelisted:

  1. Note your exact username shown in-game
  2. Contact an admin to be added
  3. Try connecting again after being added
"Invalid session" or "Not authenticated" expand_more

Java: Log out and back into the Minecraft Launcher

Bedrock: Sign out and back into your Microsoft account

Bedrock: "Unable to connect to world" expand_more
  • • Make sure port is 19132
  • • Check your internet connection
  • • Verify Microsoft account is signed in
  • • Try cellular data vs WiFi
Java: "Outdated client" or "Outdated server" expand_more
  1. Open the Minecraft Launcher
  2. Make sure you're using the latest version
  3. If server is older, create a profile for that version
