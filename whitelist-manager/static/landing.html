Current Flow

Exactly what happens when you connect

The network is whitelist-based. Approved players reach the main server. Everyone else is held in PicoLimbo until they are approved.

01

Connect

Join from Java on 25565 or Bedrock on 19132.

02

Access Check

WhitelistRouter checks whether you are approved or blocked before deciding where you go.

03

Identity

Premium Java is verified by FastLogin. Bedrock is trusted through Floodgate/XUID. Cracked Java can optionally protect itself with a password.

04

Play

Approved players reach Main. Unapproved or deactivated players are sent to PicoLimbo until an admin changes access.

family_star

What players should know

  • Premium Java: use the official launcher and go straight in if your account is whitelisted.
  • Cracked Java: if you never set a password, the server warns you every login that the name can be impersonated.
  • Protected cracked Java: if you set a password, you must use /login <password> after joining.
  • Bedrock: approved players join with Microsoft/Xbox identity and skip the Java password flow.
